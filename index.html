C language chapter one

Posted on 2019-06-08 | In C 语言学习笔记

以前上课的时候C语言没怎么认真学，现在回顾一下。

C语言出现的历史背景

C语言是在B语言的基础上发展起来的，它的根源可以追溯到ALGOL60.1960年出现的ALGOL是一种面向问题的高级语言，它离硬件比较远，不宜用来编写系统程序。

本书基本上以C89为基础并结合当前常用的C编译系统的情况进行介绍。

C语言的特点

（1）语言简洁、紧凑、使用方便、灵活。C语言一共有32个关键字，9中控制语句。
（2）运算符丰富。共有34种运算符。C语言把括号、赋值、强制类型转换等都作为运算符处理，从而使C语言的运算符类型及其丰富，表达式类型多样化。
（3）数据类型丰富。C语言提供的数据类型有：整数、浮点型、字符型、数据类型、指针类型、结构体类型、共用体内型等，能用来实现各种复杂的数据结构（如链表、树、栈等）的运算。
（4）具有结构化的控制语句。如if else语句，while语句，do whileyuju，switch语句，for语句。用函数作为程序的模块单位，便于实现程序的模块化。C语言是完全模块和结构化的语言。
（5）语法限制不太严格，程序设计的自由度较大。
（6）C语言允许直接访问物理地址，能进行位（bit）操作，能实现汇编语言的大部分功能。
（7）生成目标的代码质量高，程序执行效率高。C语言一般只比汇编语言生成的代码效率低10%~20%。
（8）比起汇编语言，可移植性好。

简单C语言程序介绍

例 1.1 输出一行信息
‘’’
‘#’ include <stdio. h>
void main()
{ print(“This is a C program. \n”);
} ‘’’
void表示此函数是空类型，不会有返回值。\n是换行符。分号是每个语句后面必不可少的。

小结：
（1）C程序是由函数构成的。一个C源程序至少且仅包含一个main函数。函数是C程序的基本单位，编写C程序就是编写一个个的函数。ANSI C提供一百多个库函数，Turbo C提供三百多个库函数。
（2）一个函数由两部分组成：
①函数的首部，即函数的第一行，包括函数名、函数类型、函数参数名、函数参数类型。
②函数体。有声明部分和执行部分。

The life

Posted on 2019-06-07

我现在已经不会表达自己的情感了。

理想很高远，却活得很苟且。

时常痛苦，为自己来到这个世界的无所作为而感到悲伤。

偶尔会和生活中的事物共鸣，也会感到快乐，也会觉得一个人并不孤独。

小的时候希望快点长大，长大后时常问自己，生命具有何等意义？却总是不能回答，也不会有正确的答案。

如果你现在问我活得好不好？我会觉得，一个人存在的意义不仅仅是对于自己。

人与人之间有一种玄妙的连接，你想跑掉，却跑不掉。

本来我应该有层次的思考，但是，又失败了。

当我这样杂乱的记录生活，生活也会更我杂乱的回应，不会更好！
